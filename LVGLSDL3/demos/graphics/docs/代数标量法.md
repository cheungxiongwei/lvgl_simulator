# 代数标量法 (Algebraic Scalar Method)

### 数学问题

已知旋转矩形的初始参数 $(O, W, H, \theta)$，其中 $O(x, y)$ 为几何中心。用户产生位移增量 $\Delta\vec{P}(dx, dy)$。求在约束单边或角点不动的情况下，新矩形的中心 $O'$ 和尺寸 $(W', H')$。

### 数学推理

1. **基向量构建：**
建立矩形的局部单位基向量：

$$\vec{e}_1 = (\cos\theta, \sin\theta), \quad \vec{e}_2 = (-\sin\theta, \cos\theta)$$

2. **位移投影：**
将世界坐标系的位移投影到局部轴，得到局部增量：

$$\delta_1 = \Delta\vec{P} \cdot \vec{e}_1 = dx\cos\theta + dy\sin\theta$$

$$\delta_2 = \Delta\vec{P} \cdot \vec{e}_2 = -dx\sin\theta + dy\cos\theta$$

3. **形变映射：**
根据拖拽方向（左/右/上/下），直接通过 $\delta$ 修改标量尺寸：
   * 拖拽左侧：$W' = W - \delta_1, \quad \Delta\xi = \delta_1/2$
   * 拖拽右侧：$W' = W + \delta_1, \quad \Delta\xi = \delta_1/2$（高度同理于 $\delta_2$）

4. **中心还原：**

$$O' = O + \Delta\xi \cdot \vec{e}_1 + \Delta\eta \cdot \vec{e}_2$$

### 注意事项

* **符号敏感：** 必须严格定义 $\delta$ 的正负号与 $W, H$ 加减的关系。
* **解耦性：** 该模型假设 $W$ 和 $H$ 的变化完全独立，在非正交变换（如剪切）下失效。
* **精度：** 仅涉及基础三角函数，舍入误差极小。

```cpp
    void update_pos(lv_point_t point) {
        if (m_item == nullptr)
            return;

        anchor_mask_t mask = get_current_anchor_mask();

        // 如果未发生旋转且拖拽的是中心点，保留原有的纯位移逻辑
        if (m_press_transform.deg == 0 || mask == ANCHOR_CENTER) {
            int dx      = point.x - m_press_point.x;
            int dy      = point.y - m_press_point.y;
            lv_area_t s = solve_rect(m_press_rect.to_aabb(), mask, dx, dy);
            sync_bbox(s);
            return;
        }

        // Step 0: 计算世界坐标系下鼠标的真实偏移量
        float dx = point.x - m_press_point.x;
        float dy = point.y - m_press_point.y;

        // 获取按下的初始状态
        float deg = m_press_transform.deg;
        float rad = deg * 3.14159265358979323846f / 180.0f;

        // 局部坐标轴基向量
        float cos_t = std::cos(rad);
        float sin_t = std::sin(rad);

        // Step 1: 将鼠标增量投影到局部坐标轴 (计算 delta_1 和 delta_2)
        float delta_1 = dx * cos_t + dy * sin_t;
        float delta_2 = -dx * sin_t + dy * cos_t;

        float new_w   = m_press_rect.width();
        float new_h   = m_press_rect.height();

        // 用于记录实际生效的变形量 (受钳位保护影响)
        float eff_d1 = 0.0f;
        float eff_d2 = 0.0f;

        // Step 2 & 3: 独立计算宽度与高度的变化，并进行尺寸退化保护钳位
        // --- 水平方向 (X轴 / e1) ---
        if (mask == ANCHOR_LEFT || mask == ANCHOR_TOP_LEFT || mask == ANCHOR_BOTTOM_LEFT) {
            new_w = m_press_rect.width() - delta_1;
            if (new_w < 1.0f) {
                new_w   = 1.0f;
                delta_1 = m_press_rect.width() - 1.0f;
            }  // 钳位保护
            eff_d1 = delta_1;  // 左侧边移动，delta_1 为正时宽度减小，中心点向右(e1正方向)偏移 delta_1/2
        } else if (mask == ANCHOR_RIGHT || mask == ANCHOR_TOP_RIGHT || mask == ANCHOR_BOTTOM_RIGHT) {
            new_w = m_press_rect.width() + delta_1;
            if (new_w < 1.0f) {
                new_w   = 1.0f;
                delta_1 = 1.0f - m_press_rect.width();
            }  // 钳位保护
            eff_d1 = delta_1;  // 右侧边移动，delta_1 为正时宽度增加，中心点向右(e1正方向)偏移 delta_1/2
        }

        // --- 垂直方向 (Y轴 / e2) ---
        if (mask == ANCHOR_TOP || mask == ANCHOR_TOP_LEFT || mask == ANCHOR_TOP_RIGHT) {
            new_h = m_press_rect.height() - delta_2;
            if (new_h < 1.0f) {
                new_h   = 1.0f;
                delta_2 = m_press_rect.height() - 1.0f;
            }  // 钳位保护
            eff_d2 = delta_2;
        } else if (mask == ANCHOR_BOTTOM || mask == ANCHOR_BOTTOM_LEFT || mask == ANCHOR_BOTTOM_RIGHT) {
            new_h = m_press_rect.height() + delta_2;
            if (new_h < 1.0f) {
                new_h   = 1.0f;
                delta_2 = 1.0f - m_press_rect.height();
            }  // 钳位保护
            eff_d2 = delta_2;
        }

        // Step 4: 计算局部中心偏移并转回世界坐标 O'
        lv_point_precise_t old_center = m_press_rect.center();

        float d_cx                    = eff_d1 / 2.0f;  // 沿 e1 方向的中心偏移
        float d_cy                    = eff_d2 / 2.0f;  // 沿 e2 方向的中心偏移

        float new_cx                  = old_center.x + d_cx * cos_t - d_cy * sin_t;
        float new_cy                  = old_center.y + d_cx * sin_t + d_cy * cos_t;

        // Step 5: 反推新矩形的轴对齐左上角坐标
        float new_x = new_cx - new_w / 2.0f;
        float new_y = new_cy - new_h / 2.0f;

        // 应用到 LVGL UI (转换为整数坐标)
        lv_obj_set_pos(m_bbox, (int32_t)std::round(new_x), (int32_t)std::round(new_y));
        lv_obj_set_size(m_bbox, (int32_t)std::round(new_w), (int32_t)std::round(new_h));
        lv_obj_update_layout(m_bbox);
        // 同步到实际的图形元素
        lv_obj_set_pos(m_item->to_object(), lv_obj_get_x(m_bbox), lv_obj_get_y(m_bbox));
        lv_obj_set_size(m_item->to_object(), lv_obj_get_width(m_bbox), lv_obj_get_height(m_bbox));

        // 更新 Pivot 始终在中心，并重新应用 Transform
        m_item->m_transform.set_pivot(new_w / 2.0f, new_h / 2.0f);

        // lv_obj_update_layout(m_bbox);
        m_item->m_transform.apply(m_bbox);
        m_item->m_transform.apply(m_item->to_object());
    }
```
