# 仿射变换分解法 (Matrix Decomposition Method)

### 数学问题

将矩形的状态表示为一个 $3 \times 3$ 的仿射变换矩阵 $M$。当用户通过锚点进行拉伸时，求一个新的矩阵 $M'$，使其在满足顶点位移的同时，保持变换的性质（如维持旋转角度不变，或允许倾斜）。

### 数学推理

**1. 统一表示：**
定义局部空间（Local Space）到一个单位正方形 $[-0.5, 0.5]$。任何矩形的状态由矩阵 $M$ 决定：
$$P_{world} = M \cdot P_{local}$$
其中 $M = T(x, y) \cdot R(\theta) \cdot S(w, h)$。

**2. 坐标空间转换：**
将鼠标的世界坐标位移 $\Delta\vec{P}(dx, dy)$ 转换到矩形的未旋转局部空间（Unrotated Local Space）：


$$\Delta\vec{P}_{local} = R(-\theta) \cdot \Delta\vec{P}_{world}$$

**3. 构造局部算子：**
在局部空间中，拉伸操作被建模为一个缩放矩阵 $S_{op}$ 和一个平移矩阵 $T_{op}$。
例如，拉动右边缘：


$$W' = W + \delta_x, \quad \text{Scale factor } s = W'/W$$

$$T_{op} \text{ 位移} = \delta_x / 2$$

**4. 矩阵复合更新：**
新的变换矩阵 $M'$ 通过左乘或右乘增量矩阵得到：
$$M' = M_{translation} \cdot M_{rotation} \cdot M_{new\_scale}$$
通过分解 $M'$，可以直接提取出新的几何中心、旋转角和宽高。

### 注意事项

* **代数严谨性：** 可以完美处理嵌套变换（如组内旋转）。

* **非均匀缩放：** 在处理旋转+拉伸时，必须确保缩放轴与旋转轴对齐，否则会产生“扭曲（Shear）”。

* **单向流动：** 所有修改都在矩阵层面完成，渲染引擎直接使用该矩阵，无需频繁转换回 $(x, y, w, h)$。

```cpp
#include <iostream>
#include <cmath>
#include <tuple>

// 简单的 2D 向量
struct Vector2 {
    float x, y;
};

// 仿射变换状态结构体
struct RectState {
    float cx, cy; // 中心点
    float w, h;   // 宽高
    float deg;    // 角度 (角度制)
};

/**
 * @brief 使用仿射变换分解逻辑计算拉伸后的矩形
 * * @param current 当前矩形状态
 * @param dx      世界坐标系 X 偏移
 * @param dy      世界坐标系 Y 偏移
 * @param mask    锚点掩码 (1:Left, 2:Right, 4:Top, 8:Bottom)
 * @return RectState 新的矩形状态
 */
RectState solve_edge_matrix(RectState current, float dx, float dy, int mask) {
    // 1. 将世界位移向量旋转到矩形的局部空间 (消除旋转干扰)
    float rad = -current.deg * M_PI / 180.0f;
    float cos_a = std::cos(rad);
    float sin_a = std::sin(rad);
    
    // 局部空间位移 (local_dx, local_dy)
    float ldx = dx * cos_a - dy * sin_a;
    float ldy = dx * sin_a + dy * cos_a;

    float dw = 0, dh = 0;
    float offset_lx = 0, offset_ly = 0;

    // 2. 在局部正交空间计算尺寸变化和中心偏移
    // 水平方向
    if (mask & 2) { // RIGHT
        dw = ldx;
        offset_lx = ldx / 2.0f;
    } else if (mask & 1) { // LEFT
        dw = -ldx;
        offset_lx = ldx / 2.0f;
    }

    // 垂直方向
    if (mask & 8) { // BOTTOM
        dh = ldy;
        offset_ly = ldy / 2.0f;
    } else if (mask & 4) { // TOP
        dh = -ldy;
        offset_ly = ldy / 2.0f;
    }

    // 3. 尺寸退化保护
    if (current.w + dw < 1.0f) {
        float adjust = 1.0f - (current.w + dw);
        // 如果是拉右边，dw 变小；如果是拉左边，dw 变大
        dw = 1.0f - current.w;
        // 重新调整位移以保持固定边不动 (此处略作简化处理)
    }
    if (current.h + dh < 1.0f) {
        dh = 1.0f - current.h;
    }

    // 4. 将局部空间的中心点偏移量旋转回世界坐标系
    float world_rad = current.deg * M_PI / 180.0f;
    float w_cos = std::cos(world_rad);
    float w_sin = std::sin(world_rad);
    
    float world_ox = offset_lx * w_cos - offset_ly * w_sin;
    float world_oy = offset_lx * w_sin + offset_ly * w_cos;

    // 5. 返回更新后的状态
    return {
        current.cx + world_ox,
        current.cy + world_oy,
        current.w + dw,
        current.h + dh,
        current.deg
    };
}

// 示例用法
int main() {
    RectState rect = {100.0f, 100.0f, 50.0f, 50.0f, 45.0f}; // 中心(100,100), 宽50, 高50, 旋转45度
    
    // 假设用户向右下角拉动了 (10, 10) 的世界坐标距离
    RectState next = solve_edge_matrix(rect, 10.0f, 10.0f, 2 | 8); // RIGHT | BOTTOM
    
    std::printf("New Rect: Center(%.2f, %.2f), Size(%.2f, %.2f)\n", 
                next.cx, next.cy, next.w, next.h);
                
    return 0;
}
```
